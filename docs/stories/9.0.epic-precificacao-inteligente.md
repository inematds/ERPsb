# Epic 9: Precificacao Inteligente (v2.0)

## Visao Geral

Ferramentas para o MEI precificar corretamente seus produtos, analisar margem real e simular cenarios. Responde a pergunta mais perigosa do pequeno negocio: "Estou vendendo com lucro ou prejuizo?"

## Problema

**90% dos MEIs precificam errado.** Compram por R$25 e vendem por R$40 achando que tem "60% de lucro". Na realidade:

```
Custo do produto:     R$ 25,00
Taxa cartao (2%):     R$  0,80
Imposto Simples (6%): R$  2,40
Custo fixo rateado:   R$  3,50  (aluguel, luz, internet / qtd vendas)
Embalagem:            R$  1,50
= Custo REAL:         R$ 33,20
Preco de venda:       R$ 40,00
Lucro REAL:           R$  6,80  (margem de 17%, NAO 60%)
```

Tres em cada dez MEIs **vendem com prejuizo** sem saber (Sebrae). A precificacao inteligente transforma isso.

## Solucao

Tres ferramentas integradas:

1. **Calculadora de preco** - "Quanto devo cobrar?"
2. **Analise de margem** - "Quanto estou lucrando em cada produto?"
3. **Simulador** - "E se eu der 10% de desconto? E se o fornecedor subir 15%?"

```prisma
  @@index([tenantId])
  @@map("custos_fixos")
}
```

## Formulas

### Preco de venda sugerido

```
custoTotal = custoProduto + embalagem + frete + outrosCustos + custoFixoRateado + imposto + taxaCartao

Onde:
  custoFixoRateado = totalCustosFixos / qtdVendasMes
  imposto = precoVenda * aliquotaSimples  (6% para comercio, 15.5% para servico)
  taxaCartao = precoVenda * taxaFormaPagamento  (2% credito tipico)

Como imposto e taxa dependem do preco, a formula e circular. Resolver com:
  precoVenda = custoBase / (1 - margemDesejada - aliquotaImposto - taxaCartao)

Onde:
  custoBase = custoProduto + embalagem + frete + outrosCustos + custoFixoRateado
```

### Margem real

```
margemReal = (precoVenda - custoTotal) / precoVenda * 100

Onde custoTotal inclui TODOS os custos (produto + fixo rateado + imposto + taxa)
```

### Markup vs Margem (educar o usuario)

```
Markup = (precoVenda - custo) / custo * 100        â†’ "Quanto coloquei em cima"
Margem = (precoVenda - custo) / precoVenda * 100   â†’ "Quanto sobra pra mim"

Exemplo: Custo R$25, Venda R$40
  Markup: 60%  (o que o MEI acha que ganha)
  Margem: 37.5% (o que realmente ganha - e isso ANTES de impostos e taxas)
```

## Metricas de sucesso

- **Adocao:** 50%+ dos usuarios ativos preenchem custo dos 5 produtos principais
- **Impacto:** Usuarios que usam precificacao tem margem media 15% maior apos 3 meses
- **Descoberta:** 30%+ descobrem pelo menos 1 produto com margem negativa
- **Acao:** 80%+ ajustam pelo menos 1 preco apos ver a analise

## Riscos

| Risco | Mitigacao |
|-------|----------|
| Usuario nao preenche custos | Campos opcionais, calcular com o que tem, incentivar gradualmente |
| Custo fixo rateado impreciso | Usar media de vendas dos ultimos 3 meses, explicar com tooltip |
| Aliquota de imposto varia | Comecar com Simples Nacional (tabela fixa por faixa), expandir depois |
| Preco sugerido "assusta" | Mostrar comparativo de mercado quando possivel |
